# 变更记录 · CHANGELOG

## 2026-06-28（二）— 配色升级 + 体验打磨

### 配色（采用方案 C · 现代石墨 + 翠绿）
- 全站换为中性灰白 + 清亮翠绿（emerald）主色，阴影中性化；深浅色都更清晰醒神。
- 新增背景纹理 token `--dot`（极淡网点）+ `--glow`（顶部柔光），克制不花哨。
- 品牌标与 favicon 换成清单勾选图标（替掉「题」字占位）。

### 交互
- 答错后新增「重新作答 ↻」：练习模式可重选并重新确认，提交按更正后答案计分。
- 横屏 / 全尺寸下「上一题 / 下一题」改为 48px 方形大按钮，触控更舒服。

### 修复
- 题目导航网格：固定 5 列、强制正方形单元格、消除横向溢出与「堆叠 bug」观感（`align-content/align-self:start` + `overflow-x:hidden`）。

## 2026-06-28（一）— 完整重构 + 设计系统化

### 设计系统
- 新增 `docs/design-system.md`：颜色 / 字体 / 间距 / 组件 / 交互 / 响应式的唯一规范。
- 新增 `CLAUDE.md`：项目硬性约束（单文件可部署、变量化样式、单引擎、深浅色、多端）。
- 刷新 `README.md`：补全新功能说明与项目结构。

### 视觉
- 全新护眼配色（首版为暖纸白 + 静谧绿，后于「二」升级为石墨 + 翠绿）。
- 颜色全部 token 化（`:root` + `[data-theme="dark"]`），右上角**手动切换深浅色**，记忆选择、首次跟随系统、预涂防闪烁。
- 重做首页套题卡、题卡、成绩页（环形正确率仪表 + 评级）。

### 交互
- **题目导航抽屉**：题号网格总览（未答 / 已选 / 对 / 错配色），跳题、跳到标记、直接交卷。
- **键盘快捷键**：`1–9` 选项、`←/→` 翻题、`Enter` 确认/下一题、`F` 标记、`R` 显答案、`Esc` 关导航。
- **触控手势**：题卡左右滑动切题。
- **单题标记**（`⚑`）。
- **进度自动保存**：刷新 / 退出后自动恢复，可从头开始。
- 主按钮状态机：确认 → 下一题 → 提交；未答交卷二次确认。

### 适配
- 手机竖屏 / 平板 / 桌面 / 横屏统一响应式；吸顶 header + 吸底操作栏；安全区适配；热区 ≥44px；输入框 ≥16px 防 iOS 缩放；`prefers-reduced-motion` 降级。

### 离线版
- 重写为与主站同款设计 + 同一份引擎，离线同样支持深浅色 / 导航 / 键盘 / 滑动 / 进度恢复 / 重做错题。

### 修复
- 修复导航遮罩层（`.scrim`）关闭时仍铺满全屏、拦截点击导致整页无法操作的问题（未激活时 `pointer-events:none`）。
- 清理深色主题中重复的 `--surface` 声明。
